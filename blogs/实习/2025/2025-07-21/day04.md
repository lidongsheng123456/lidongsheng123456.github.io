## 四.基础学习

### 1.Rust有一个约定优于配置的原则

1. src/lib.rs 》库的根模块；
2. src/main.rs 》 二进制程序的根模块；
3. src/bin/*.rs 》 多个二进制程序。

### 2.Rust关键词完整列表

1. 声明和定义：fn - 定义函数 let - 变量绑定 mut - 可变性标记 const - 常量定义 static - 静态变量 struct - 结构体定义 enum - 枚举定义 union - 联合体定义 trait - 特征定义 impl - 实现块 type - 类型别名
2. 控制流：if - 条件语句 else - 否则语句 match - 模式匹配 loop - 无限循环 while - 条件循环 for - 迭代循环 break - 跳出循环 continue - 继续下次循环 return - 函数返回
3. 模块和可见性：mod - 模块定义 use - 导入项目 pub - 公共可见性 crate - 当前包 super - 父模块 self - 当前模块实例 Self - 当前类型
4. 内存和所有权：move - 强制移动 ref - 引用模式 unsafe - 不安全代码块 extern - 外部函数接口
5. 异步编程：async - 异步函数 await - 等待异步操作
6. 其他：where - 泛型约束 as - 类型转换 in - 用于for循环 dyn - 动态分支

### 3.元组与数组的区别

|特性|元组（Tuple）|数组（Array）|
| --------| --------------------| ------------------|
|类型|可以包含不同数据类型|只能包含相同类型|
|大小|编译时确定，不可变|编译时确定，不可变|
|访问方式|索引访问Tuple.0|索引访问array[0]|
|语法|（T1,T2,T3）|[T,N]|
|内存布局|连续存储，可能有填充|连续存储，紧密排列|
|用途|组合不同数据类型|存储同类型数据集合|

### 4.整数数据类型的 有符号整数 or 无符号整数区别

1. 有符号整数（i8...）:表示负数到正数，适合需要负数1的场景；
2. 无符号整数（u8...）：只有非负数，适合计算索引大小。

### 5.闭包和普通函数的区别以及闭包的优势

```rust
// 闭包定义如示例：
let x = 1;
let multiply = | x:i32,y:i32 | x * y + x;
println!("闭包计算{}",multiply(4,6)); 
```

### 6.if判断以及if表达式

```rust
// 例
let flag = true
let result = if flag {5} else {6}; // 5
```

此时result值为5，特点是条件判断可省略圆括号，后面表达式花括号里面的如果是表达式可以省略return，直接返回但返回的值必须是rust的数据类型，如果当前语句还有else分支，那么else里面返回的值类型要跟if里面一样。

### 7.for循环的参数传递

```rust
// 例
for i in 1..=5{
    println!("{}",i)
}
```

i代表变量名，i是每次迭代后的数据 ，in代表要迭代的范围，1..=5代表从1开始到5停止，如果是1..5代表从1开始到4停止，没有等于号的可以理解为左闭右开，取不到最右边的值，i不需要初始值，初始值为1，因为 in 右侧写了迭代范围，初始值就是 i 的起始值。

### 8.while与loop循环

while同java一样，如果传true则死循环，传表达式则为false时停止执行，loop没有判断的地方，是个标准的死循环，需要做好逻辑判断来使用条件语句 if 写入break退出循环，例如：

```rust
let mut sum = 0;
loop {
    sum += 1;
    if sum == 5 {
        break sum * sum;    
    }
}
```

### 9.match语句的使用

### 10.Rust中双冒号的用法

1. 在导入语句中：use rust_demo::{ 函数 } 这里的双冒号代表路径分隔，用来分隔模块的层级关系；
2. 在代码层面中：String::from() 这里的双冒号表示调用String这个类中的公共函数。

### 11.Rust中的数据类型

#### 11.1基本数据类型（不触发所有权转移，因为赋的是值）

1. 所有整数类型：i8，i16，i32，i64，i128，isize，u8，u16，u32，u64，u128，usize；
2. 浮点类型：f32（单精度），f64（双精度）；
3. 布尔类型：bool；
4. 字符类型：char；
5. 元组（如果所有元素都是copy类型（基本类型））；
6. 数组（如果所有元素都是copy类型（基本类型））。

#### 11.2引用类型（会触发所有权转移，因为赋的是地址不是值）

1. String；
2. Vec<T>；
3. HashMap<k,v>；
4. 使用struct自定义结构体。

### 12.所有权规则

1. Rust的所有权规则是其安全性和内存管理的基础；
2. 所有权是一个抽象的概念，用于管理内存安全，并不是一个引用地址；
3. 基本数据类型和引用数据类型都有所有权，但是基本数据类型进行赋值时会赋值值而不是移动所有权，引用类型赋值会移动所有权；
4. 基本数据类型不会触发所有权转移，原变量不会被回收，引用类型会触发，因为赋值时赋的是地址。

### 13.小结

1. 不可变变量可以不显式声明数据类型，让rust自动推断，但是也可以选择显式声明类型以提高代码清晰度或指定特定类型；
2. 所有变量的定义都要加let，lei修饰的变量可以不指定类型，由rust自动根据赋值的数据进行推断，const常量的声明必须指定数据类型；
3. 函数的形参类似于ts的类型限制，箭头右边定义着返回的数据类型，如果函数体里面是表达式且没有加分号则不需要return，如果需要提前返回值，可以加return提前返回，函数里面如果有多次逻辑处理，但最下面只要有表达式且没有加分号，就可以不加return。

```rust
// 例如
fn add(a: i32,b: i32)-> i32{
    let sum = a+ b;
    sum + 0
}
// 这段逻辑即使有多次计算，但只要最后一条是表达式，就可以省略分号，直接返回数据。
```
